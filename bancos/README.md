# Bancos de palabras

Cada archivo de esta carpeta corresponde a un juego.

Ejemplo:

- `ahorcado-bank.js` controla el banco de Ahorcado.
- `memory-bank.js` controla el banco de Memory Game.

Edita solo el archivo del juego que quieras actualizar.
No borres esta parte del inicio:

```js
window.ESL_WORD_BANKS = window.ESL_WORD_BANKS || {};
window.ESL_CATEGORY_LEVELS = window.ESL_CATEGORY_LEVELS || {};
```

Agrega palabras respetando el formato existente.
