Cómo agregar palabras sin tocar código gigante

En cada juego usa el editor interno que ya existe. Cuando presiones Descargar archivo actualizado, el archivo se descargará con el nombre correcto para reemplazarlo en GitHub.

1. Entra al juegoAbre Anagrama, Ahorcado, Memory, Word Match o Word Search.
2. Abre ConfigurarUsa el menú lateral del juego y entra al modo edición.
3. DescargaPresiona “Descargar archivo actualizado” luego de aplicar los cambios.
4. Reemplaza en GitHubSube el HTML descargado en la carpeta juegos y confirma el cambio.
Hecho para organizar tus juegos ESL en GitHub Pages.
