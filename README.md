# ESL Games 2

Proyecto organizado con una página de inicio y tus juegos HTML originales.

## Estructura

- `index.html`: menú principal.
- `juegos/anagrama.html`
- `juegos/ahorcado.html`
- `juegos/memory.html`
- `juegos/word-match.html`
- `juegos/word-search.html`

## Cómo agregar palabras

1. Abre el juego desde el menú.
2. Entra a Menú → Configurar → Modo edición / código avanzado.
3. Agrega las palabras en el nivel correspondiente.
4. Pulsa **Descargar archivo actualizado**.
5. Sube el archivo descargado a GitHub dentro de la carpeta `juegos`, reemplazando el archivo del mismo nombre.
6. Haz **Commit changes**.

No edites el HTML gigante manualmente si puedes evitarlo. Usa el editor interno de cada juego.
